## Labels to Add

Create these labels for better issue management:

- `enhancement` - New feature or request (green)
- `bug` - Something isn't working (red)
- `documentation` - Improvements or additions to documentation (blue)
- `kiroween` - Related to Kiroween hackathon (purple)
- `good first issue` - Good for newcomers (green)
- `help wanted` - Extra attention is needed (yellow)
